Lasso Regression: ecuacion lineal

Lasso ajusta una combinacion lineal de las variables numericas para estimar PM 2.5. Antes de entrenar, cada variable se escala con StandardScaler (media 0, desviacion 1), por eso los coeficientes de abajo no se aplican directamente sobre los valores originales del formulario, sino sobre la version escalada de cada variable.

Lasso agrega una penalizacion L1 a la funcion de costo tradicional de regresion lineal: 

costo = error_cuadratico_medio + alpha * suma( |coeficiente| )
alpha = {{ metrics_data.lasso_equation.alpha or 0 }}

Esa penalizacion puede llevar coeficientes exactamente a cero, lo que equivale a que el modelo descarte una variable por completo. En este entrenamiento, Lasso dejo en cero {{ metrics_data.lasso_equation.variables_en_cero or 0 }} variable(s).

Decision Tree Regressor: como calcula su prediccion

El arbol no ajusta una formula continua como Lasso. En cada nodo prueba cortes del tipo variable <= umbral y elige el que mas reduce el error cuadratico medio ({{ metrics_data.tree_math.criterion or "squared_error" }}) entre los dos grupos resultantes. Este proceso se repite hasta llegar a las hojas finales.

La primera pregunta que hace el arbol (nodo raiz) es {{ metrics_data.tree_root_feature or "PM 10" }} <= {{ "%.2f"|format(metrics_data.tree_root_threshold or 0) }}, evaluada sobre las {{ "{:,}".format(metrics_data.tree_math.root_samples or 0) }} filas de entrenamiento.

La prediccion final de cualquier hoja es simplemente el promedio de PM 2.5 de las filas de entrenamiento que terminaron en ese grupo. Por ejemplo, antes de hacer ningun corte, el promedio general (prediccion del nodo raiz) es {{ "%.2f"|format(metrics_data.tree_math.root_prediction or 0) }}.
